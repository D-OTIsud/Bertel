# Archive

This folder contains retired documents that were removed from active repository surfaces
during conservative cleanup.

Policy:
- Runtime SQL/API behavior is not changed by archived docs.
- Archived items are considered historical snapshots.
- Full original content remains recoverable from git history.

Archived categories:
- `reports/`: time-bound audit/performance snapshots
- `legal/`: historical legal integration/refactoring guides superseded by unified schema docs
- `docs/`: superseded doc-matrix artifacts
