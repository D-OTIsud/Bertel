Fonctionnalités & portée

Implémentation surtout CSS/config, sans changement de données. Lève d'un coup la dette transverse a11y / perf / thème.

Ce que ça corrige

  • S1 Focus : règle globale :focus-visible {outline:2px solid var(--ring)} + suppression des outline:none non remplacés.
  • S7 Contraste : --muted #66767d → #5c6a71 (5.1:1) ; retrait de #94a1a8 sur texte.
  • S8 Mouvement : garde-fou @media (prefers-reduced-motion: reduce) ; easing ease-out-expo (fin du bounce).
  • S5 Perf : migration vers next/font (fin du @import bloquant) ; fond dégradé non fixed.
  • S6 Thème : une seule famille de tokens (suppression des doublons --orange-soft/--accent-soft, des 2 échelles radius/ombre).
  • Distinctions (retour PO) : cocarde de classement .thumb__rating (étoiles, ÷2, à cheval sur la photo) + classement en toutes lettres .class-text dans le corps (pas d'icônes étoiles redondantes), labels .label-logo / .label-chip (sans pastille d'état ; un label non valide n'est pas affiché), tags .tag-chip (couleur via --tag), et le signet .thumb__labels à cheval sur la miniature. Classement / labels / tags conservés sur cartes et fiches.

Fichiers impactés

  • src/styles.css (:root tokens, focus, motion, contraste)
  • src/features/object-editor/object-editor.css (focus/motion/tokens locaux)
  • src/app/layout.tsx (next/font/google)

Critères d'acceptation

  • Tab au clavier : anneau visible sur boutons, chips, nav, champs.
  • Lighthouse a11y ≥ 95 ; contraste AA sur muted/labels.
  • OS « réduire les animations » → aucune transition > 0.
  • Aucun token dupliqué ; build sans Google Fonts externe.
  • Classement / labels / tags restent visibles sur cartes et fiches ; la miniature des fiches classées ou labellisées porte un signet à cheval (cocarde de classement + pastilles de label).
  • Seuls les HEB portent un classement (étoiles / clés / épis), montré en cocarde sur la photo et en toutes lettres dans le corps. Le statut d'un label vit dans l'éditeur §08 ; côté public, seuls les labels valides s'affichent, sans pastille d'état.
