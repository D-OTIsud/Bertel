Fonctionnalités & portée

Comportement par type (créneau méta)

  • HEB : statut ouvert · chambres · capacité · gamme prix
  • ITI : distance · D+ · durée · forme · difficulté (pas de pastille ouvert)
  • FMA : prochaine/plage de dates · commune · billetterie
  • RES : statut ouvert · cuisine · couverts · gamme prix
  • VIS : accès · nature du site · commune
  • SRV/SPU : accès · horaires · PMR

Règles

  • La pastille « ouvert/fermé » ne s'affiche que pour HEB/RES/SRV (sens horaire) ; jamais pour ITI/FMA/VIS.
  • Couleur de la pastille de type = palette d'archétype unique (impl. 2.1).
  • États : hover (relief), focus (anneau), sélection (★), squelette au chargement.

Classement · labels · tags (retour PO)

  • La carte conserve le classement, les labels et les tags : ils restent visibles dans le corps via .distinction-row (étoiles .stars, puces .label-chip avec pastille de statut accordé/en cours/expiré, tags colorés .tag-chip + chip « +N » de débordement).
  • La miniature porte un signet à cheval : cocarde de classement (.thumb__rating) en haut-gauche, pour HEB uniquement (étoiles · clés · épis) ; logos-pastilles de label (.thumb__labels .label-logo) en bas-droite pour les objets labellisés.
  • Pas d'étoiles de classement sur RES/ASC/ITI/VIS/FMA/SRV ; seul l'hébergement est classé par étoiles.

Données & fichiers

  • ResultCardView.tsx : remplacer le bloc méta fixe par un rendu par archétype.
  • utils/explorer-card.ts : projeter les champs méta selon le type (déjà présents côté filtres).
  • Pastille via type-pill acc-* de 2.1 ; statut ouvert via open_now conditionné par type.

Critères d'acceptation

  • Chaque type affiche sa ligne méta dédiée (snapshot par type).
  • Aucune pastille ouvert sur ITI/FMA/VIS.
  • Carte cliquable au clavier, anneau de focus visible.
  • Le classement, les labels et les tags d'un objet sont préservés sur la carte (jamais supprimés à la projection).
  • Une miniature d'objet classé ou labellisé porte un signet à cheval : cocarde d'étoiles (HEB seul) et/ou logos-pastilles de label, sans rogner le bord de la carte.
