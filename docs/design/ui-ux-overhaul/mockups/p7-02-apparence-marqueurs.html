Fonctionnalités & portée

7.2 livre deux panneaux « Plateforme » du hub 7.1. Apparence édite la marque white-label (palette + logo + aperçu en direct, appliqués globalement via les tokens de thème). Marqueurs carte passe d'une pile de 7 cartes toutes déployées (environ 50 contrôles à l'écran) à une liste maître plus un panneau d'édition unique, avec le SVG personnalisé replié derrière « Avancé ».

Comportement

  • Apparence : palette (primaire, accent, texte, fond, surface) + logo, avec aperçu en direct, appliqués globalement via les tokens de thème. Réinitialiser restaure les valeurs par défaut.
  • Marqueurs : liste des 7 types à gauche, panneau d'édition d'un seul type à droite (couleur + icône preset). Un type modifié à la fois, plus de mur de contrôles.
  • SVG personnalisé derrière une divulgation « Avancé » (<details>) : import ou collage, puis « Appliquer ». Assaini par liste blanche, sans script ni contenu externe.
  • Les deux panneaux sont gated super-admin : les non-admins voient un aperçu en lecture seule plus la raison du verrouillage.
  • Branding et marqueurs sont enregistrés ensemble depuis ce panneau.

Données & fichiers

  • src/views/SettingsPage.tsx : la section marqueurs passe à un agencement maître/détail.
  • src/config/map-markers.ts : objectTypeOptions, sanitizeCustomMarkerSvg, buildMarkerDataUri.
  • src/services/branding.ts : saveBrandingSettings.
  • Theme store (palette/logo) + ui store (markerStyles).
  • Branding et marqueurs sont persistés ensemble à l'enregistrement.

Critères d'acceptation

  • Un seul type de marqueur est édité à la fois (liste maître + panneau unique).
  • La zone de texte SVG est masquée derrière la divulgation « Avancé ».
  • L'aperçu en direct reflète la palette (en-tête en dégradé, pastilles, hex).
  • La réinitialisation fonctionne (palette et marqueurs reviennent aux valeurs par défaut).
  • :focus-visible visible au clavier sur les pastilles de couleur, les tuiles d'icône et les résumés details.
  • Gating super-admin appliqué ; état lecture seule plus raison pour les autres rôles.
